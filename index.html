Project Overview

Methodology

01

Causal Feature Extraction

Rather than raw correlations, we extract causal edges from GDELT event streams using structural causal models. Features that survive causal screening remain stable under distribution shift — a key advantage over standard factor models.

02

RL Agent Training

A PPO agent trained on causal features learns a directional policy for weekly SPY positioning. VIX term structure acts as a regime gate — the agent only trades in low-uncertainty windows, dramatically improving Sharpe.

03

Rigorous OOS Validation

All reported metrics are strictly out-of-sample, walk-forward, no lookahead. Straddle and options overlays tested separately — directional signal confirmed real, theta decay problem isolated and documented.
